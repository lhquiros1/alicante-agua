# Proyecto: Experiencia del agua en Alicante

## Objetivo
Crear una experiencia visual vertical (9:16) donde una gota se transforma en un personaje y recorre el sistema del agua.

## Importante
- NO es un mapa
- NO gráficos tradicionales
- Estilo orgánico, fluido

## Tecnología
- PixiJS
- Canvas único

## Escena

1. Gota aparece
2. Se mueve en vertical
3. Se transforma
4. Recorre sistema
5. Ciclo continuo

## Datos

Usar data/datos.json para:
- consumo mensual
- porcentajes
- intensidad

## Reglas

- No representar GIS literal
- Convertir datos en comportamiento visual
